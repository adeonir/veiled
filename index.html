Clean Time Machine Backups, Effortlessly

Veiled automatically excludes development artifacts from Time Machine. It scans your projects, identifies build directories and dependencies, and keeps exclusions updated with a daily daemon.

$ brew install adeonir/tap/veiled
$ veiled start
Daemon activated. Running initial scan...
Scanned 47 projects in ~/Developer
Excluded 312 directories (18.4 GB saved)
~600 KB Binary size
7+ Stacks supported
0 Dependencies

Everything You Need, Nothing You Don't

One binary. Zero configuration. Every stack.

Smart Detection

Three layers: .gitignore rules per repository, built-in list covering 7+ ecosystems, and your own custom exclusions.

Set and Forget

A native launchd daemon runs daily at 3 AM. New projects are covered automatically without manual intervention.

Multi-Stack

JavaScript, Python, Rust, Go, Swift, Java, PHP. Every major ecosystem's build artifacts and dependencies are covered.

Auto-Update

Checks GitHub Releases on each run. Downloads, verifies SHA-256 checksum, and replaces the binary atomically.

Backup-Safe

Only affects future backups. Existing Time Machine snapshots are never touched. Veiled manages its own exclusions separately from yours.

Fully Reversible

One command removes all managed exclusions and restores your system to its original state. No leftovers.

How It Works

01

Install

One Homebrew command. A single ~600 KB native binary with zero runtime dependencies.

brew install adeonir/tap/veiled
02

Start

Activate the launchd daemon. Veiled runs an initial scan and registers itself to run daily.

veiled start
03

Forget

That's it. Veiled scans your projects every day, excludes new artifacts, and updates itself in the background.

Full Control When You Need It
